# AnalogYouTube

## Основной функционал

### Видео
- Загрузка видео
- Получение списка видео
- Просмотр видео
- Обработка видео (сжатие, конвертация через FFmpeg)

### Чат
- Общение между пользователями
- Реалтайм через WebSocket
- Система друзей (запросы на добавление)
- Приватные чаты (общение с друзьями после принятия запроса)

### Донаты
- Отправка донатов
- История транзакций

### Обязательный функционал
- Система лайков
- Подписки на авторов
- Рекомендательная система видео
- Авторизация и аутентификация пользователей

### Архитектура проекта

Проект реализован с использованием трёхслойной архитектуры:
- **Controller (Handler)**: Принимает HTTP-запросы, валидирует данные, возвращает ответы.
- **Service**: Содержит бизнес-логику (видео, донаты, чат, рекомендации и т.д.).
- **Repository**: Работает с базой данных (CRUD операции).


### Преимущества:
- Чистая структура проекта
- Лёгкость тестирования
- Масштабируемость

### Технологический стек
- Язык: Go
- Архитектура: Controller / Service / Repository
- HTTP роутинг: Gorilla Mux
- WebSocket: Gorilla WebSocket
- Видео обработка: FFmpeg
- База данных: PostgreSQL

### REST API Эндпоинты

### Пользователи
- `POST /api/v1/register` — Регистрация нового пользователя
- `POST /api/v1/login` — Авторизация и получение токена (JWT)
- `GET /api/v1/users/{name}` — Получение профиля пользователя
- `PUT /api/v1/users/{name}` — Обновление данных профиля
- `PUT /api/v1/users/{id}` - 

### Видео
- `GET /api/v1/videos` — Получение списка видео (с пагинацией)
- `POST /api/v1/videos` — Загрузка нового видео
- `GET /api/v1/videos/{id}` — Получение информации о конкретном видео
- `PUT /api/v1/videos/{id}` — Обновление информации о видео (название, описание)
- `DELETE /api/v1/videos/{id}` — Удаление видео

### Взаимодействие
- `POST /api/v1/videos/{id}/like` — Поставить лайк видео
- `DELETE /api/v1/videos/{id}/like` — Убрать лайк с видео
- `POST /api/v1/users/{id}/subscribe` — Подписаться на автора
- `DELETE /api/v1/users/{id}/subscribe` — Отписаться от автора

### Донаты
- `POST /api/v1/donations` — Отправка доната автору
- `GET /api/v1/users/{id}/donations` — Получение истории транзакций пользователя

### Друзья
- `POST /api/v1/users/{id}/friends/request` — Отправить запрос в друзья
- `POST /api/v1/users/{id}/friends/accept` — Принять запрос в друзья
- `DELETE /api/v1/users/{id}/friends/reject` — Отклонить запрос / Удалить из друзей

### Чат
- `GET /ws/chat/{videoId}` — Подключение к WebSocket для реалтайм чата во время трансляций/премьер
- `GET /api/v1/chats` — Получить список активных приватных чатов
- `GET /ws/chat/private/{chatId}` — Подключение к приватному чату с другом через WebSocket
